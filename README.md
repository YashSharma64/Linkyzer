
# Linkyzer

AI-powered LinkedIn profile auditor that predicts selection score, detects fake or weak experiences, and suggests projects based on skills, college year, and domain.

## Features
- Profile Selection Possibility Score
- Fake/Weak Experience Detection
- AI Suggested Projects
- Startup/Problem Solver Personality Insight
- Smart Recruiter-style Summary
- Downloadable AI Report (PDF)

## Tech Stack
Frontend: React / Next.js  
Backend: Node.js / Express (or Flask)  
AI: OpenAI / Gemini API  

## Setup
cd frontend → npm install  
cd backend → npm install
